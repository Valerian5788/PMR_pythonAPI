Documentation de l'API CharleroiTri

Cette API retourne les données de gestion de foule pour la gare de Charleroi pour une journée spécifique.

Utilisation

Utilisez l'URL suivante pour accéder aux données :

/CharleroiTri/<jour>/

Remplacez <jour> par la date au format ddmmyy (par exemple, 311023 pour le 31 octobre 2023).

Exemple de réponse

[
    {"timestamp": "2023-02-02T02:00:00", "zone": "Gare de Charleroi", "total": 275},
    {"timestamp": "2023-02-02T03:00:00", "zone": "Gare de Charleroi", "total": 375},
    {"timestamp": "2023-02-02T04:00:00", "zone": "Gare de Charleroi", "total": 389},
    
]

Les données sont renvoyées au format JSON et comprennent le timestamp, la zone (qui est toujours "Gare de Charleroi" dans ce cas) et le total de personnes pour chaque heure.
