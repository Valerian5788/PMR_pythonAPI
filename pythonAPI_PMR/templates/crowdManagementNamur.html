Documentation de l'API NamurTri

Cette API retourne les données de gestion de foule pour la gare de Namur pour une journée spécifique.

Utilisation

Utilisez l'URL suivante pour accéder aux données :

/NamurTri/<jour>/

Remplacez <jour> par la date au format ddmmyy (par exemple, 020223 pour le 02 février 2023).

Exemple de réponse

[
    {"timestamp": "2023-02-02T00:00:00", "zone": "Gare de Namur", "total": 227},
    {"timestamp": "2023-02-02T01:00:00", "zone": "Gare de Namur", "total": 232},
    {"timestamp": "2023-02-02T02:00:00", "zone": "Gare de Namur", "total": 215},
    
]

Les données sont renvoyées au format JSON et comprennent le timestamp, la zone (qui est toujours "Gare de Namur" dans ce cas) et le total de personnes pour chaque heure.
