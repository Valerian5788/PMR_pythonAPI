Documentation de l'API getFacilitiesOfATrain

Cette API retourne les installations disponibles pour un train spécifique.

Utilisation :

Envoyez une requête GET à l'URL suivante :

/jsonComposition/{id}/

où <id> est l'identifiant du train (sans IC, seulement le chiffre).
